# 🛒Interactive Power BI dashboard for analyzing E-commerce sales performance using data visualization and business intelligence.

## 📌 Project Overview

This project is an interactive **E-Commerce Sales Dashboard** developed in **Power BI** to analyze sales performance across different states, customers, product categories, and payment modes. The dashboard enables users to monitor key business metrics and identify trends through interactive visualizations.

---

## 🎯 Objectives

- Analyze overall sales performance
- Track profit and sales amount
- Monitor quantity sold
- Compare sales across different states
- Identify top customers and profitable sub-categories
- Analyze payment mode preferences

---

## 🛠️ Tools & Technologies

- Power BI
- Power Query
- DAX

---

## 📊 Dashboard Features

- KPI Cards (Total Amount, Quantity, Profit, Average Order Value)
- Sales by State (Map)
- Monthly Profit Analysis
- Quantity by Category
- Top Customers by Sales
- Quantity by Customer
- Profit by Sub-Category
- Payment Mode Distribution
- Interactive Filters (State, Sub-Category, Quarter)

## 📈 Key Insights

- Total Sales Amount: **438K**
- Total Quantity Sold: **6K**
- Total Profit: **37K**
- Average Order Value: **121K**
- COD is the most preferred payment method.
- Clothing category generated the highest quantity sold.
- Monthly profit trends help identify peak and low-performing months.
- State-wise sales analysis highlights regional business performance.
